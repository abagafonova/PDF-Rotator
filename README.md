## Use case: OCR of scanned book photos

This tool is especially useful when you have PDFs made from photos of books taken sideways to speed up scanning. Rotating the PDF first produces a properly-oriented file that OCR tools can process more accurately.

Workflow:

2. Load `book_rotated.pdf` into your OCR tool
